# 邀请返利

邀请好友使用 NextToken，每当被邀请的人首次充值消费后，邀请人会立即获得 **20%** 的返利。返利会直接发到邀请人的账户余额里。

## 如何找到邀请链接

1. 前往 [个人资料页面](https://www.nextoken.online/profile)。
2. 在页面底部找到邀请返利的相关链接和邀请码。
3. 复制邀请链接或邀请码分享给好友。

好友通过邀请链接注册并首次消费后，返利会自动发到邀请人的账户。

## 返利规则

- 被邀请的人首次充值消费后，邀请人会立即获得**充值金额的20%**。
- 返利直接发到账户余额，可用于后续消费。

## 相关入口

- [个人资料](https://www.nextoken.online/profile) — 查看和管理邀请链接
- [计费说明](https://www.nextoken.online/docs/platform/billing.md) — 了解计费方式和购买选项

## 注意事项

- **禁止刷单**:同一人多账号、使用机器人批量注册的"下线"会被取消返利,严重的封禁推广人账号
- **禁止误导宣传**:推广文案不得虚假承诺(如"绝对不限速""所有模型免费"等)。我们鼓励真实可信的推广
- **以页面规则为准**:具体返利比例、结算周期、最低提现额可能调整,以「邀请」页实时显示为准
