# 使用政策

本文档说明 NextToken 的使用规则，适用于 NextToken 网站、API、API Key 以及相关接入方式。

## 概述

您不得使用 NextToken 从事违法、侵权、欺诈、骚扰、恶意攻击、绕过限制或其他滥用平台资源的行为。

对于部分模型、线路或功能，可能适用额外的使用限制。NextToken 会在对应产品页、文档或控制台中进行说明；使用相关能力即表示您同意遵守对应说明。

## 适用范围

本政策适用于以下场景：

- 使用 NextToken 网站、控制台或 API
- 创建、存储、管理、配置或使用 API Key
- 通过 Claude Code、Codex、CC-Switch、OpenCode 或其他客户端接入 NextToken
- 基于 NextToken 提供自动化流程、机器人、接口或其他功能

通过兼容客户端、自有程序或其他集成方式接入 NextToken 本身不被禁止，但不得借此绕过 NextToken 的鉴权、配额、速率限制、安全保护或产品说明。

## 禁止的使用行为

### 非法或侵权行为

不得使用 NextToken 从事、协助或推广任何违法或侵权活动，包括但不限于：

- 诈骗、洗钱、赌博、非法集资或其他非法活动
- 侵犯知识产权、商业秘密或其他合法权益
- 传播依法被禁止的内容

### 欺诈、冒充与误导

不得使用 NextToken 生成、传播或自动化输出以下内容：

- 冒充个人、机构、客服、官方身份的内容
- 用于诈骗、社工、钓鱼、诱导转账或虚假承诺的内容
- 明知不实仍用于误导他人的内容
- 批量垃圾内容、垃圾营销或骚扰信息

### 攻击、绕过与资源滥用

不得使用 NextToken 进行或协助进行以下行为：

- 入侵、扫描、爆破、撞库、钓鱼、提权或其他恶意探测行为
- 编写、传播或优化恶意代码、木马、勒索软件或其他破坏性程序
- 针对任何系统实施 DDoS、资源消耗攻击或其他破坏性行为
- 绕过 NextToken 或相关服务的鉴权、配额、速率限制或安全保护
- 通过异常调用、空转请求、刷量或其他方式恶意消耗平台资源

### 账户与风控规避

不得：

- 借用、盗用、冒用他人账户或支付手段
- 通过批量注册、虚假身份、脚本化方式规避平台限制或风控
- 将 API Key 向无授权第三方转让、出售、出借或公开披露

### 隐私与数据权益侵害

不得：

- 非法收集、出售、公开或滥用他人个人信息
- 使用泄露数据、非公开信息或其他无合法来源的数据实施识别、跟踪或骚扰
- 生成或处理侵犯他人隐私、肖像、名誉或身份权益的内容

### 有害内容

不得使用 NextToken 创建、传播或协助生成以下内容：

- 宣扬自残、自杀、虐待、极端暴力或恐怖活动的内容
- 仇恨、歧视、骚扰、羞辱或针对特定群体的人身攻击内容
- 性剥削、未成年人相关不当内容或其他严重有害内容

### 高风险使用限制

不得将 NextToken 作为唯一依据，直接用于影响个人重大权益的决策，包括但不限于医疗、法律、金融、保险、招聘、教育录取、住房资格以及执法等场景。

如您在相关场景中使用 NextToken，应自行确保人工复核、必要披露以及符合法律法规要求。

## API Key 与账户安全要求

您应妥善保管账户和 API Key，并仅在经授权的成员、服务器或集成环境中按最小必要原则使用。

您应采取合理措施保护 API Key，包括但不限于：

- 不将 API Key 写入公开仓库、公开客户端或其他可被第三方直接获取的位置
- 按用途区分不同密钥，避免长期混用
- 发现疑似泄露后及时轮换、停用或删除相关密钥
- 按公开文档和产品说明接入和调用服务

文档内容不构成：

- 对任何第三方产品、模型或平台的额外授权
- 对绕过限制、规避风控、批量滥用或攻击行为的许可

您不得将本站文档、示例或接入说明用于冒充官方认证、虚构合作关系或误导他人认为某种接入方式已获得额外授权。

## 数据与隐私

- 您通过 API 发送的请求内容,**仅用于完成本次调用**,不用于训练模型

- 不要在 prompt 中输入超出业务必要范围的敏感信息(身份证号、密码、医疗记录等)

更详细的数据处理说明请参考主站「隐私政策」。
